"A blockchain project: A decentralized reputation system where users can mint skill-based NFT badges by completing verifiable challenges or getting endorsements from other users. Each badge represents a specific skill (coding, design, writing, etc.) and contains metadata about the achievement level and verification method. Users can stake STX tokens to endorse others' skills, and successful endorsements earn rewards while false endorsements result in slashing. The system includes a marketplace where users can showcase their badge collections, and employers or collaborators can verify credentials on-chain. Smart contracts handle badge minting, staking mechanics, reputation scoring, and a simple governance system for adding new skill categories."
